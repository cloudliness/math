# MathFlow — AI Discrete Math Tutor

MathFlow is a fully local, RAG-powered interactive study assistant designed to help students learn Discrete Mathematics. By leveraging LlamaIndex, ChromaDB, and powerful frontend rendering libraries, it can parse uploaded textbook chapters, answer complex questions with context, and generate interactive flowcharts, node graphs, and mathematical plots.

This project was built to explore dynamic LLM interfaces that go beyond basic text generation.

## 🌟 Features

- **Retrieval-Augmented Generation (RAG)**: Upload any Discrete Math PDF. The backend uses `LlamaParse` and `HuggingFace` embeddings to chunk and ingest the text into a local `ChromaDB` vector database.
- **Interactive UI Components**: When the AI explains concepts like state machines, trees, or algebraic functions, it can output targeted JSON data to render:
  - **Node Graphs & Trees** via [React Flow](https://reactflow.dev/).
  - **Coordinate and Algebraic Plots** via [Mafs](https://mafs.dev/).
  - **Complex Math Notation** via KaTeX and React Markdown.
- **Document Management**: View all uploaded library documents in the sidebar. **Toggle** specific documents on or off to restrict the AI's context window to specific chapters, or permanently delete them from the vector DB.
- **Chat History**: Chat sessions are saved locally to JSON files so you can refer back to previous sessions.

## 🛠️ Technology Stack

**Frontend**
- React 19 / Vite
- React Flow (Node-based UI visualization)
- Mafs (React-based math coordinate system visualization)
- React Markdown / remark-math / rehype-katex

**Backend**
- Python 3.10+ / FastAPI
- LlamaIndex (RAG orchestration & Context Retrieval)
- ChromaDB (Persistent Vector Store)
- OpenRouter API (Using `nousresearch/hermes-2-pro-llama-3-8b` by default)
- LlamaParse (High-quality PDF markdown extraction)
- HuggingFace Embeddings (`BAAI/bge-large-en-v1.5`)

## 🚀 Quick Start

### 1. Prerequisites 

You will need API keys for the LLM and PDF parsing. Create a `.env` file in the `backend/` directory:

```env
OPENROUTER_API_KEY="your-openrouter-key"
LLAMA_CLOUD_API_KEY="your-llamacloud-key"
```

### 2. Run the Application

If you are on Linux or macOS, you can quickly start both the frontend and backend servers together using the provided shell script:

```bash
./start_servers.sh
```

To stop them:

```bash
./stop_servers.sh
```

### 3. Manual Start

**Backend (FastAPI)**
```bash
cd backend
source venv/bin/activate
python3 -m app.main
```
The API runs at `http://localhost:8000`.

**Frontend (React/Vite)**
```bash
cd frontend
npm install
npm run dev
```
The UI runs at `http://localhost:5173`.
